# WellMom Logo

## Lokasi Logo

Simpan file logo WellMom dengan nama:
- `logo-wellmom.png`

Di folder ini: `public/assets/images/`

## Spesifikasi Logo

- Format: PNG dengan background transparan
- Ukuran minimum: 200x200 px
- Ukuran yang disarankan: 512x512 px
- Warna primary WellMom: #3B9ECF (biru)

## Jika Logo Belum Tersedia

Sementara waktu, Anda bisa menggunakan placeholder atau membuat logo sementara dengan:
1. Background biru (#3B9ECF)
2. Ikon ibu dan bayi (sesuai branding WellMom)
3. Text "WellMom" dengan font yang clean dan modern
